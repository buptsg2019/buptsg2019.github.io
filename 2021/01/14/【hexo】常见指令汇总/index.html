hexo常见指令汇总

| 指令 | 简写 | 功能 |
| - | - |
| hexo init 文件名 | | 新建hexo文件夹 |
| hexo generate | hexo g | 生成页面文件 |
| hexo server | hexo s | 启动服务预览 |
| hexo serser -p 端口号 |hexo s -p 端口号 | 指定端口运行 |
| hexo deploy | hexo d | 部署 |
| hexo new ‘文章标题’ | | 新建markdown博客文件 |
| hexo generate -draft | hexo g -d | 将草稿部署到github |
| hexo clean | | 清除缓存 |
| hexo version | hexo v | 显示hexo版本 |
